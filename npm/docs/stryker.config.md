# stryker.config.mjs

## Огляд

Цей файл конфігурує середовище для виконання тестів за допомогою Vitest. Він визначає механізми виконання тестів, збору звітів про покриття та мутації, а також налаштовує інкрементальний режим. Код спирається на конфіги: mutation.json, incremental.json.

## Поведінка

1. Визначає Vitest як засіб виконання тестів.
2. Вказує конфігураційний файл для Vitest.
3. Налаштовує аналіз покриття на рівні кожного тесту для прискорення.
4. Встановлює тимчасову директорію для звітів Stryker.
5. Визначає репортери для виведення результатів у форматі JSON та звичайному тексті.
6. Вказує ім'я файлу для збереження JSON-звіту про мутації.
7. Вмикає інкрементальний режим для збереження результатів між запусками.
8. Вказує ім'я файлу для збереження інкрементальних даних.
9. Додає локальний плагін для ігнорування мутацій Vue `<script setup>`-макросів.
10. Вказує ігнорувачі для мутацій, зокрема `vue-macros`.
