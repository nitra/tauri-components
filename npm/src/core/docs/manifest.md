# manifest.js

## Огляд

Модуль реалізує функції для роботи зі специфікаціями інструментів. Він перетворює специфікації інструментів у формат JSON Schema. Модуль генерує повний маніфест інструментів у форматі OpenAI та створює спрощені списки інструментів, що містять їхні імена та описи.

## Поведінка

toJsonSchema перетворює специфікацію вхідних даних інструменту на об'єкт JSON Schema.
toolManifest створює масив визначень інструментів у форматі OpenAI, фільтруючи їх за наданим предикатом.
listTools створює компактний список інструментів, що містить лише їхні імена та короткі описи.

## Публічний API

toJsonSchema — перетворює опис вхідних даних інструменту на об'єкт JSON Schema.
toolManifest — надає визначення інструментів для виклику функцій OpenAI, з можливістю фільтрації.
listTools — виводить стислий перелік доступних інструментів (назва та короткий опис).
