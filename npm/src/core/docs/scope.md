# scope.js

## Огляд

Модуль визначає та керує дозволами акторів на використання інструментів. Він встановлює рівні акторів за допомогою `DEFAULT_ACTOR_TIERS`, визначає доступні інструменти для кожного актора через `scopedToolNames` та надає маніфест дозволів за допомогою `scopedManifest`.

## Поведінка

DEFAULT_ACTOR_TIERS визначає максимальний рівень виконання для різних типів акторів.
classify визначає, чи дозволено актору викликати інструмент, чи потрібне схвалення, чи заборонено.
scopedManifest повертає перелік інструментів, які актор може виконати або для яких потрібне схвалення.
scopedToolNames повертає список назв інструментів, які актор бачить.

## Публічний API

DEFAULT_ACTOR_TIERS — Визначає рівні доступу для акторів.
classify — Визначає, до якого актора належить виклик інструменту.
scopedManifest — Перелік інструментів, доступних актору для виконання або запиту дозволу.
scopedToolNames — Список назв інструментів, видимих актору.
