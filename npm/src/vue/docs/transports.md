# transports.js

## Огляд

Модуль ініціює виклики зовнішніх інструментів через Tauri. Він передає визначені команди та необхідні їм аргументи, а потім повертає отриманий результат виконання.

## Поведінка

1. Викликає команду Tauri, використовуючи визначення інструменту.
2. Передає вхідні дані інструменту як аргументи відповідній команді.
3. Повертає результат виконання команди.
