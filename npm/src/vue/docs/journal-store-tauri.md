# journal-store-tauri.js

## Огляд

Цей модуль керує доступом до механізму журналу. Він дозволяє створювати сховище журналу та отримувати повний список існуючих записів.

## Поведінка

1. Викликати `createTauriJournalStore` для ініціалізації сховища журналу, яке маршрутизує операції з файловою системою через команди `tauri-plugin-agent`.
2. Використовувати отримане сховище для створення нового запису журналу, викликаючи команду `plugin:agent|journal_create`.
3. Використовувати отримане сховище для завантаження запису журналу за його ідентифікатором, викликаючи команду `plugin:agent|journal_load`.
4. Використовувати отримане сховище для оновлення запису журналу за його ідентифікатором, застосовуючи патч, викликаючи команду `plugin:agent|journal_update`.
5. Використовувати отримане сховище для отримання списку всіх записів журналу, викликаючи команду `plugin:agent|journal_list`.

## Гарантії поведінки

- Read-only: не виконує операцій запису (ФС/БД).
