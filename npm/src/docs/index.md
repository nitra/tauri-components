# index.js

## Огляд

Створює та ініціалізує повний набір компонентів для роботи агента. Відповідає за зв'язування циклу агента, поверхні інструментів та каталогу застосунків. Керує диспетчеризацією запитів та відповідей, забезпечуючи кешування результатів у межах одного прогону. Забезпечує взаємодію з чатом OpenAI та налаштування механізму роботи з моделлю OMLX.

## Поведінка

1. Створює набір компонентів для агента, що зв'язує цикл агента, поверхню інструментів та каталог застосунків.
2. Обробляє схвалення застосунку.
3. Обробляє запит.
4. Обробляє відповідь.
5. Створює механізм диспетчеризації та перевіряє вхідні дані.
6. Створює інтеграцію з чатом OpenAI та запускає агента.
7. Перелічує моделі OMLX.
8. Вирішує модель OMLX.
9. Визначає базовий URL OMLX, включаючи прямий та проксі-URL, з можливістю кешування.
10. Перелічує інструменти, створює схему JSON для інструментів та отримує маніфест інструментів.
11. Класифікує елементи та визначає рівні акторів за замовчуванням, включаючи обмежений маніфест та імена інструментів у межах області дії.
12. Отримує визначений інструмент.

## Гарантії поведінки

- Read-only: не виконує операцій запису (ФС/БД).
- Кешує результати в межах одного прогону.
