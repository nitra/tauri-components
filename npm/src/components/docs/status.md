# status.js

## Огляд

Файл визначає кольорові значення, що відповідають різним станам запиту. Публічні функції `STATUS_COLOR` та `statusColor` повертають конкретний акцентний колір для заданого статусу або сірий колір у випадку невідомого статусу.

## Поведінка

STATUS_COLOR — Надає об'єкт, що містить визначені кольори для різних станів запиту.
statusColor — Повертає відповідний акцентний колір для заданого статусу запиту, використовуючи сірий колір за замовчуванням для невідомих статусів.

## Публічний API
