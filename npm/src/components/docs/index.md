# index.js

## Огляд

Компоненти цього файлу керують відображенням діалогових вікон, які взаємодіють з агентом. Експортується базовий клас для діалогів, компоненти для відображення специфічних дій, деталей запиту та поточного стану, а також константи для визначення кольорів стану.

## Поведінка

1. Експортує компонент `AgentDialog` для відображення діалогових вікон, що взаємодіють з агентом.
2. Експортує компонент `AuditDialog` для відображення діалогових вікон аудиту, що взаємодіють з агентом.
3. Експортує компонент `BaseDialog` як базовий елемент для створення діалогових вікон.
4. Експортує компонент `DialogActions` для відображення дій у діалогових вікнах.
5. Експортує компонент `RequestView` для відображення деталей запиту.
6. Експортує компонент `StatePill` для відображення стану.
7. Експортує константи `STATUS_COLOR` та `statusColor` для визначення кольорів стану.
