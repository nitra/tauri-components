# StatePill.vue

Компонент відображає візуальний маркер стану (pill). Він використовується для індикації статусу в різних частинах застосунку.

## Поведінка
1. Компонент отримує текстовий статус.
2. Компонент визначає відповідний колір на основі отриманого статусу.
3. Компонент відображає текстовий статус.
4. Компонент відображає візуальний маркер (крапку) кольору, визначеного для цього статусу.

## Гарантії поведінки
* Відображення завжди містить текстовий статус.
* Колір візуального маркера та фону компонента коректно відображається відповідно до наданого статусу.
