# RequestView.vue

Компонент відображає детальний вигляд результату виконання запиту. Він візуалізує статус, резюме, помилки та список виконаних дій.

## Поведінка

1. Відображає статус результату за допомогою компонента `StatePill`.
2. Якщо у результаті є резюме, відображає його як текстовий блок.
3. Якщо у результаті є питання, відображає його як текстовий блок.
4. Якщо у результаті є помилка, відображає її як червоний текстовий блок.
5. Якщо у результаті є список дій, відображає їх у вигляді окремих рядків.
6. Для кожної дії відображає іконку, що вказує на успіх або помилку виконання дії.
7. Для кожної дії відображає назву інструменту та його вхідні дані.

## Гарантії поведінки

* Компонент завжди відображає статус результату, оскільки властивість `result` є обов'язковою.
* Текстові блоки (резюме, питання, помилка) відображаються лише за наявності відповідного поля у `result`.
* Список дій відображається лише за наявності елементів у масиві `actions` у `result`.
* Іконка для кожної дії коректно відображає стан виконання дії (успіх або помилка).
