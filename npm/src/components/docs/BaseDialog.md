Компонент надає універсальну структуру для діалогових вікон у застосунку. Він інкапсулює типовий макет діалогу, включаючи заголовок, тіло та кнопки дій.

## Поведінка

1. Відображає діалогове вікно, кероване станом `modelValue`.
2. Відображає заголовок, що складається з іконки (за наявності), назви (`title`) та кнопки закриття.
3. Дозволяє вставити контент у тіло діалогу через основний слот.
4. Дозволяє вставити елементи керування (кнопки) у нижню частину діалогу через слот `actions`.
5. При зміні стану `modelValue` зовнішнім компонентом, компонент емітує подію `update:modelValue` з новим значенням.

## Гарантії поведінки

* Діалогове вікно завжди має визначену назву (`title`).
* Ширина діалогового вікна за замовчуванням становить 520 пікселів, але може бути перевизначена.
* Компонент коректно передає всі додаткові атрибути, отримані від батьківського елемента, до базового елемента діалогу.
