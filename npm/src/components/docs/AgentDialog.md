Компонент відображає діалогове вікно для взаємодії з локальною мовною моделлю (LLM) через агент. Він дозволяє користувачеві вводити запити, налаштовувати конфігурацію агента та бачити історію чату.

## Поведінка

1. **При відкритті діалогу (`onShow`):**
    * Очищається поле введення та історія чату.
    * Ініціалізується конфігурація агента, завантажуючи налаштування з глобальних даних.
    * Завантажується список доступних моделей від сервера LLM.
2. **При зміні конфігурації:**
    * Користувач може вказати базовий URL, вибрати модель або ввести API ключ.
3. **При введенні запиту та натисканні "Запустити"/"Надіслати":**
    * Запит знімається з поля введення.
    * Запис користувача додається до історії чату.
    * Статус "думає" активується.
    * Викликається метод агента для виконання запиту (новий запит або продовження існуючої розмови).
    * Після отримання відповіді від агента:
        * Відповідь агента додається до історії чату.
        * Статус "думає" деактивується.
        * Історія чату автоматично прокручується до останнього повідомлення.
4. **При завершенні операції:**
    * Якщо операція була успішною, компонент може повідомити зовнішній контекст про завершення роботи агента.

## Гарантії поведінки

* Діалогове вікно відображає історію чату у порядку чергування: повідомлення користувача, а потім відповідь агента.
* Під час обробки запиту (відправлення або отримання відповіді) інтерфейс блокується, і відображається індикатор "думає...".
* При відсутності введення або активній обробці запиту, кнопка відправки вимкнена.
* При відкритті діалогу, список моделей заповнюється асинхронно, відображаючи стан завантаження.
